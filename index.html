Q1 function counter(){
            var counter = 0;
        
            function IncreaseCounter() {
                return counter += 1;
            };
        
            return IncreaseCounter;
        }
        
        var counter = counter();
        alert(counter());
        alert(counter());
        alert(counter());
        alert(counter());

        output // 1 2 3 4 
        when you call counter function it will return IncreaseCounter then
        when IncreaseCounter will call it will return counter += 1
        counter += 1 stored in the var counter 
        so a an output it will keep incrementing the counter and alert the number 



        Q2 let count = 0;
        (function () {
          if (count === 0) {
            let count = 1;
            console.log(count); // What is logged?
          }
          console.log(count); // What is logged?
        })();

        output // 1  0

        it is a self invoking function and the functional scope is created
        initial value of counter is 0 so the if condition will be true and it will allocate a count an memory in functional scope and assign it wil 1 
        so the first console.log will print 1 
        and second console.log will print 0 
        as the count=1 is limited to functional scope so it will take the global value which is 0
